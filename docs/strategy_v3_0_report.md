## 四、防御层内部高抛低吸机制

防御层（红利低波 50% + 国债 50%）每周固定比例。当红利低波上涨时，
防御层资金自动从红利低波减仓至国债；当下跌时则反向加仓。

**无需动态调整，50:50 自然实现了低买高卖。**

> 结论：阈值 0~10% 影响极小（Sharpe 波动 <0.003），当前 7% 合理。

## 九、风险提示

1. 所有因子计算通过 `src/factors.py`（ddof=0），禁止自行实现
2. 日频 DD 比周频高约 0.3~2pp，需周内监测
3. 纳指ETF 如出现 QDII 溢价 >2%，需人工判断
4. 国债ETF 为场内品种，需关注折溢价
5. 历史回测不保证未来收益
