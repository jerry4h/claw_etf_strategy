# 虾池ETF轮动策略 v3.0

基于 **5只ETF** 的周频动量轮动策略，全连续/零门控/三层架构。
**Sharpe 1.311 / 年化 14.38% / 最大回撤 7.52%**（2013-05 ~ 2026-07）。

## 策略原理

### 三层架构

| 层次 | 决策 | 方法 |
|------|------|------|
| **Layer 1** 买什么 | 进攻层选 TOP2 | `score = mom4 − 1.05×vol20`（mom_w=1 固定） |
| **Layer 2** 买多少 | 进攻层权重分配 | inv-vol12（波动率倒数加权，ddof=0） |
| **Layer 3** 防多少 | 进攻 vs 防御比例 | 纳指 vol20 ∈ [15%, 35%] → 防御 [25%, 95%] 线性插值 |

> 相比等权持仓（TOP2 各 50%），inv-vol12 提升 Sharpe **+9.9%**（1.311 vs 1.193），累计终值 +8.3%。

### 最终参数

| 参数 | 值 | 安全边界 | 说明 |
|------|:--:|:--------:|------|
| `mom_w` | **1.0** | 固定 | 动量权重（固定） |
| `vol_w` | **1.05** | 0.80~1.20（MC=100%） | 波动率惩罚权重 |
| `inv_vol_window` | **12** | ≥8 安全，≤5 MC 降至 17% | 波动率倒数平滑窗口 |
| `step_low` | **0.15** | 0.12~0.20 安全 | 防御起效的 vol 下限 |
| `step_high` | **0.35** | 0.25~0.45 安全 | 极限防御的 vol 上限 |
| `def_alloc` | **0.25** | 0.20~0.35 安全 | 基准防御比例 |
| `max_def` | **0.95** | 未扫描边界 | 极限防御比例 |
| `max_single_alloc` | **0.40** | 0.35~0.50 零影响 | 单只 ETF 权重上限 |
| `rebalance_threshold` | **7%** | 0%~10% 波动 <0.003 Sharpe | 调仓触发阈值 |
| `fee_rate` | **0.005%** | 单边 | 交易费率 |

### 调仓阈值的影响

| 阈值 | Sharpe | 年化 | DD | 交易次数 |
|:---:|:-----:|:---:|:--:|:------:|
| **0%**（每周调） | **1.313** | 14.3% | 7.6% | 648 |
| 1% | 1.312 | 14.3% | 7.5% | 648 |
| 2% | 1.311 | 14.3% | 7.5% | 647 |
| **7%**（当前） | **1.311** | 14.4% | 7.5% | 642 |
| 10% | 1.312 | 14.4% | 7.4% | 636 |

> **结论**：阈值 0~10% 的 Sharpe 波动 <0.003，**几乎无影响**。当前 7% 已能充分捕捉调仓信号，且全年多交易 <0.5 次。

### 防御层内部均衡（50:50 红利低波 vs 国债）

每周防御层内部强制均衡为 50:50，不依赖调仓阈值，效果显著提升：

| 方式 | Sharpe | 年化 | DD |
|------|:-----:|:---:|:--:|
| ✅ **每周再平衡** | **1.607** | 15.10% | 6.58% |
| ❌ 阈值触发 7% | 1.311 | 14.38% | 7.52% |

> 防御层内 50:50 每周强制均衡，天然实现了红利低波与国债间的高抛低吸。
> 红利低波上涨→超配→下期均衡时减仓至国债；红利低波下跌→买入更多→等待反转。
> 全回测 0.005% 费率下有效，零费率仅再提升 0.01~0.02% 累计收益。

## 如何运行

### 单次回测
```bash
cd /home/ubuntu/claw_etf_strategy
python scripts/run_backtest.py --config config/strategy_v3_0_final.yaml
```

### 实时调仓计算
```bash
python scripts/rebalance_live.py
```
输出下周一持仓方案，含 Layer 1~3 分解。

### 超参优化
```bash
python scripts/optimize_v3_0_newdata.py
```

### 鲁棒性评估
```bash
python scripts/run_robustness.py --config config/strategy_v3_0_final.yaml \
  --output output/robustness_final/ --n-mc 400 --oat --n-wf 9
```

## 注意事项

- 数据列顺序：`日期,纳指ETF,红利低波ETF,沪深300ETF,黄金ETF,国债ETF`
- **所有因子计算强制使用 `src/factors.py`（ddof=0）**，禁止自行实现
- 如需多日频分析，日频 DD 比周频高约 0.3~2pp
- 纳指ETF如出现 QDII 溢价 >2%，需人工判断是否延迟买入
- 防御层 50:50 每周均衡已验证有效，可通过 `scripts/weekly_rebalance.py` 启用
- 策略基于历史回测，不保证未来收益
