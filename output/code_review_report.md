## 一、策略概要

基于 5 只 ETF 的周频动量轮动策略，四层架构：

| 层次 | 决策 | 方法 |
|------|------|------|
| Layer 1 买什么 | 进攻层选 TOP2 | score = mom4 - 1.10*vol11 |
| Layer 2 买多少 | 进攻层权重 | inv-vol10 波动率倒数加权 |
| Layer 3 防多少 | 进攻 vs 防御 | 纳指 vol11 线性插值 [25%, 95%] |
| DefAlloc 防什么 | 红利低波 vs 国债 | hl_ratio = clip(0.80 - 2.67*vol_hl, 0, 0.80) |

资产池: 纳指ETF(513100) / 中证500ETF(510500) / 黄金ETF(518880) / 红利低波ETF(512890) / 国债ETF(511010)

## 四、方法论评估

### 4.1 优势

- **四层架构逻辑清晰**: 选股、权重、防御、防御内配各层独立，易于理解和调试
- **全连续零门控**: 防御比例随 vol 线性变化，避免离散跳变带来的参数悬崖
- **因子计算集中管理**: src/factors.py 是唯一的 ddof=0 因子源，杜绝重复实现
- **参数安全边界明确**: 每个参数都有 MC/OAT 验证的安全区间
- **DSR 极强**: 即使 n_trials=50（远超实际探索量），DSR 仍为 1.0，alpha 真实性高
- **成本鲁棒**: 20bp 单边成本下仍有正 alpha

### 4.2 已确认的风险

| 风险 | 严重度 | 说明 |
|------|--------|------|
| WF 相对胜率偏弱 | 🟡 | 5/9 窗口跑赢等权，策略并非在所有市场环境下都优于被动持有 |
| 资产池后视选择 | 🟡 | 沪深300→中证500 迁移基于回测表现，存在 universe selection bias |
| 防御信号单一 | 🟡 | Layer 3 仅依赖纳指 vol，A 股独立下跌时可能延迟响应 |
| 全区间回测非独立验证 | 🟡 | 参数和资产池在同一数据集上选择，缺乏纯样本外验证 |

### 4.3 已排除的风险

| 原担忧 | 验证结果 |
|--------|----------|
| DSR n_trials 低估导致过拟合 | n_trials=50 时 DSR 仍为 1.0，过拟合风险极低 |
| 交易成本侵蚀 alpha | 20bp 下 Sharpe=1.357，不敏感 |
| 参数 clamp 错误导致鲁棒性虚高 | 已修复(1a56ba0)，MC=100% 在正确范围下仍成立 |

## 十、Kimi 审计报告整合验证（2026-07-23）

基于 Kimi 机构级审计报告的建议，补充以下验证实验：

### 10.1 ddof=0 vs ddof=1 敏感性

| ddof | Sharpe | 年化收益 | 最大回撤 |
|------|--------|----------|----------|
| 0（当前） | 1.514 | 15.98% | 7.33% |
| 1（金融学惯例） | 1.527 | 15.80% | 7.55% |

**结论**: ddof=1 使 Sharpe 变化 +0.013，影响极小（<0.05）。ddof=0 的选择对策略无实质影响，Kimi 报告中标记的"风险A"可降级为 P3（无需修改）。

### 10.2 防御层消融实验（Brinson 归因简化版）

| 配置 | Sharpe | 年化收益 | 最大回撤 |
|------|--------|----------|----------|
| 完整策略 (L3+L4) | 1.514 | 15.98% | 7.33% |
| 禁用 L3（固定25%防御） | 1.323 | 15.54% | 8.87% |
| 禁用 L4（固定50/50防御） | 1.400 | 15.92% | 12.03% |
| 禁用 L3+L4 | 1.269 | 15.60% | 10.90% |

**独立贡献**:
- Layer 3（vol 动态防御）: Sharpe +0.191, DD 压缩 1.54pp
- Layer 4（DefAlloc 红利/国债切换）: Sharpe +0.115, DD 压缩 4.70pp

**结论**: Layer 3 是 Sharpe 的核心贡献者（+0.191），Layer 4 是回撤压缩的核心贡献者（-4.70pp）。两层均有显著独立价值，验证了四层架构设计的合理性。
